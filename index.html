Interaction Hub

Developmental Roadmap for One-Button Users

Visual Stimulation:
Phase 1: Sensory Agency
Goal: Understand "I press, and something happens." Pure cause and effect.
🎧DJ Party 🫧Bubble Pop 🎨Neon Canvas
Phase 2: Sustained Action
Goal: Repeated pressing moves an object through space.
🐍Mega Snake 🏎️Neon Racer
Phase 3: Persistence
Goal: Multi-step process with a delayed high-value reward.
🚀Mega Rocket 👾Monster Feed
Phase 4: Timing (Action on Cue)
Goal: Press only when the world is ready. Teaches waiting and inhibitory control.
🪵Mega Log Hop
Phase 5: Visual Matching
Goal: Choice making. Select the icon that matches the target.
🎯Target Match
Phase 6: Logic & Categorization
Goal: Identify groups. Sort by size, color temperature, or category.
📦Logic Sort
Phase 7: Pre-Communication
Goal: Using scanning to request items or express a state.
